# Songs Hub

Search and Play your favorite artist's albums

Tech Stack:

- React
- JavaScript

In the project directory, you can run:

-`npm install` -`npm start`

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

To run the test:

- `npm run test`
